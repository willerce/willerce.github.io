从 Willog 到 Noderce

2012年08月11日

自从换掉 Wordpress 以后，就注定要挖各种各样的坑，用 C# / Nancy 写的 Willog 因为 Godaddy 太慢，没有合适的 Win 主机等各种原因早已经有了用别的语言重写的想法。于是就有了 Noderce

新手入门，挑着简单、资料多的来，使用了 Express 框架，直接用 JSON 格式文件做存储，各外，优雅的 Markdown 支持是不可少的。

###一点点说明

  • /storage/ 所有的 json 文件都保存在这个目录里
  • /storage/user.json 保存着用户名密码，存储的密码是用 MD5 加密一次，初始用户名密码都是： admin
  • /storage/timeline.json 是所有文章的数组，文件的前后排序也靠它控制
  • /storage/post 目录里放的是所有文章
  • /storage/page 目录里放的是自定义页面，比如：关于、链接之类的页面
  • /storage/recycle-bin 目录是回收站，当你的文章别名改掉的时候，旧的 JSON 文件会被移到这里来，亲，给加个 一键清理回收站 功能呗？

###运行

只需装好nodejs环境，命令： node app.js 即可运行

Github Repository URL : https://github.com/willerce/noderce
