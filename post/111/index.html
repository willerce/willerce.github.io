《使用HTML制作商业网站》课程结束

2009年11月25日

11月23日，《使用HTML制作商业网站》这门课程结束了，24号参加了中心的HTML内测。取得了比较好的成绩。

当然，作为ACCP5.0 S1阶段的课程，它和上一本书《使用Java语言理解程序逻辑》一样，涉及到的都是各自所在领域比较基础的内容，比如标签，表格，框架，Dreamweaver的使用等等。本课程有稍微提到了DIC+CSS，DIV层只是在DreamWeaver中拖拖拉拉，而CSS也只在Dreamweaver进行可视化编辑。

这一课程的理论课与上机练习中，大量地使用了淘宝网与拍拍网的素材,可能这是全书唯一和书名商业网站相呼应的地方了，要是只以书中学到的浅薄技术制作商业网站，除非企业老板是傻子才可能要你的案例。所以建议在ACCP6.0将书目将一下吧？当然，这些知识都是基础，熟练的使用它们是非常重要的。

在学习这个课程时,在课程教材的基础上,另外看了一些书目.推荐给大家.


  • 《别具光芒DIV+CSS网页布局与美化》李烨 著 / 人民邮电出版社

  • 《CSS Mastery: Advanced Web Standards Solutions》 Andy Budd … 著 / FriendSoft / links


  • 有中文版本《精通CSS:高级WEB标准解决方案》陈剑瓯 译/ 人民邮电出版社




另推荐几个个人认为比较好的HTML的教程:


  1. 网页设计师:循序渐进 http://www.w3cn.org/article/step/index.html

  2. 蓝色理想,Div+CSS布局入门教程 http://www.blueidea.com/tech/site/2006/3574.asp


PS：还有一些其它的书目，资料，如有兴趣，可向我索取，另，以上提到的书，可以使用电驴找到它们。
