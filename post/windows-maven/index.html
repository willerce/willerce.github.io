Windows，Maven的安装与环境配置

2011年09月27日

Apache Maven 是一个软件项目管理和理解工具，更多请看： Maven介绍

###安装步骤：


  1. 安装配置JDK，设置%JAVE_HOME%，参见： Java Development Kit(JDK)下载及环境设置

  2. 下载 Apache Maven ，解压到一个目录。（我使用的是：D:\maven\apache-maven-3.0.3）

  3. 添加系统变量 %M2_HOME%

  4. 将 %M2_HOME%\bin 添加到 path 变量中

Win 7打开系统环境变量步骤：我的电脑->空白处右键->属性->高级系统设置->环境变量->（添加相关变量）

###步骤图：

最后，按 WIN+R ，输入 CMD 运行，敲命令： mvn -version，见到以下信息，即安装成功。
