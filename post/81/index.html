2009-02，流水帐

2009年02月26日

生活总是很现实，于是我很现实的生活了一段时间，忍不住，又来网络上发发牢骚，写写我最近的忙的。


  1. 每天24小时中，大部分时间都在市场，这可能成为我今后的主要工作。继续浪费时间中。

  2. 准备购买本本，现用ASUS的 Eee Pc，不错，但把我的眼睛近视度数又加了，都是小屏幕惹的祸。
    现在准备入手的有两款本本，Dell studio 1537，还有就是Acer AS 6930G 741。用过的朋友告诉我一下，这两款机子怎么样。最近经常泡在本友会，本友会，真不错。 benyouhui.it168.com

  3. 前段时间接了一个网站，朋友介绍的，是一个在俄罗斯这边销售木制玩具的公司要做个产品网站。三月份之前要完成它，已经快完成了。

  4. 一直想专心的学学俄语，如果能在地铁里和俄国人聊聊天，那该多好，但我静不下心。


最近很烦，很多事都不顺。乌克兰这个国家现在经济状况很差，乌克兰是市场皮衣的主要销售对象，现在这样子，皮衣没人要了。生意很差啊，现在摊位费，午饭都贵了。要人命啊。
