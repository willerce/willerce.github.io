Noderce 停止开发

2014年03月13日

非常抱歉，因为时间的关系，我不能继续开发 Noderce 了。

刚开始这个项目的时候，刚从 Wordpress 换到自己写的 C# 博客博客，正好在学习 Node.JS 就重写了一个，代码写得很草，也没有什么规划。很开心有同学愿意用这个小程序，也提交了一些 PR。这是一个非常小众的程序，虽然后台简陋，没有漂亮的主题，也没有插件支持，但写写文章还是足够的。

目前没有已知的 Bug，所以你可以继续使用这个程序，除非她不再满足你的需求。

我选择使用 Hexo 这个静态博客程序，并使用 Github Page 托管博客。目前你看到的，就是这个方案。这里有一个脚本，可以把文章导出为 Markdown 文件，评论导出为多说的数据格式。

Noderce at Github: https://github.com/willerce/noderce
