回想当初学习Discuz!及介绍EasyAPM-Serve

2008年06月13日

06年9月份加入了学校的青橄榄文学社，担任“设计”一职，所谓“设计”就是排版。而负责老师这样说：“除了封面，要尽量少用图，文学是一种艺术，太花俏会影响到内容。“ 这位老师的话不是本文重点。重点是，当时的我，想利用这一学校的唯一“媒体”，创建一个学校的网上聚居点。于是，在这样的条件下，我开始去了解、学习、使用Discuz ! 论坛程序。

Discuz !（下文简称DZ）是一个相当成熟、易用的论坛程序。许多非专业性的论坛站长对PHP，HTML等了解甚少，但却搭建起了一个人气相当旺的论坛，这在于DZ的易用性与丰富的第三方插件与模版，同时Discuz! support team也出品了相当多的作品，现在我也是Discuz! support team风格组的成员之一。

如果你对Discuz ! 有兴趣，你可以到Discuz官方论坛看看，也可以进入 Discuz! support team 论坛瞧瞧！

记得去年DST还是想当的火。当时值DZ 6.0发布之际，团队里面临了许多的插件、风格升级任务。在那段时间，我成长了许多，在些感谢DST的 缘亦如此、小七、水月、还有团队的大大，伊泽浩。

而我在升级风格过程中，需要本地测试，于是翻出了EasyAPM-Serve，其实当时并不知道他的名字。这个名字我也是今天才听到，我当时用的比较旧的版本可以，在看到EasyAPM-Serve出口人是micronsky.net时才敢确定。而micronsky.net是这个环境套件的Mysql密码。现在micronsky.net已经关闭，进入后是一个广告停靠页，请不要访问。

介绍一下EasyAPM-Serve:
EasyAPM-Server 是 Win2000(XP/2003) 系统下的 Apache+PHP+MySQL 运行环境快速集成安装套件，是使用开源及免费软件组合的预配置服务器运行环境，适合服务环境快速建立，本套件是免费软件，遵循 GPL (GNU General Public License) 协议，所有包含软件下载于官方网站，无任何后门或广告程序.
包含以下程序：
Apache v2.0.55 (Win32)
PHP v5.0.5 (Win32)
Zend Optimizer v2.5.10a (Win32)
MySQL v5.0.15 (Win32)
phpMyAdmin v2.6.4-pl4
AcrivePerl v5.8.7.815

下载地址在文未，下载后，双击安装，不要改变它的安装路径。安装完成后它将自动开启服务。
目录简要说明：

+–\tmp\　　　　　　　 临时文件目录
|
+–\local\　　　　　　 程序目录
|　|

|　|
|　+–\mysql4\　　　　 MySQL 程序目录
|　|　|

|　|

MySQL 数据库初始帐密:用户名: root ；密码: micronsky.net ，安装后请使用 phpMyAdmin 修改密码。
安装完成后，WWW目录下的PHPMYADIMN文件不要删除。可以进入http://127.0.0.1/phpmyadmin 进行表创建，密码修改等操作。
