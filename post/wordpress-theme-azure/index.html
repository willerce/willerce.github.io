WordPress Theme：Azure

2012年03月23日

主题移植至：Windows Azure，这是一款单栏主题，底部含有一些私人的链接，右浮条同样为私人社交链接，所以你要自己手动修改它们。期望有朋友来做一个后台设置这些链接的功能（感谢 @francis 实现了此功能）。

第一次将主题托管在 Github 上，如果你有兴趣，欢迎一起来维护，你需要自己动手，才能更好的使用它。Have fun ^_^

GitHub Repositories : https://github.com/willerce/wordpress-theme-azure

更新日志:

  • 2012-04-01: 感谢 @francis 添加了增加了后台设置功能，可以去右侧及询问链接进行设置。可以在后台自定义Google搜索ID（add social setting of theme right side，update admin settings, add feed and ga settings）

其它版本:

  • Deef 制作了 Typecho 版本，在这里获取。
  • Sawyer 制作的修改版 Fake-Azure V2.0 For WordPress
