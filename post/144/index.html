北大青鸟福州三拓（福州博洋教育）的骗局：“Oracle WDP课程”

2010年08月28日

北大青鸟福州三拓在2010年06月份左右开始，在学员当中开始宣传WDP课程，作为北大青鸟ACCP课程的福州校区点，福州三拓为什么抛弃北大青鸟ACCP课程，又为什么选择Oracle的WDP课程？

什么是WDP？
Oracle WDP 全称为Oracle Workforce Development Program（Oracle职业发展力计划）, 是Oracle （甲骨文）公司专门面向学生、个人、在职人员等群体开设的职业发展力课程，是Oracle公司针对职业教育市场在全球推广的项目，其以低廉的成本给这部分人群提供Oracle技术培训，经过系统化的实训，让这部分人群能够迅速掌握Oracle最新的核心技术，并能胜任企业大型数据库管理、维护、开发工作。

参与Oracle WDP学习可以获取的证书
参加完WDP课程学习，由Oracle公司组织学员统一考试，考试通过学员可获取Oracle公司颁发的数据库工程师OCP证书。

主要学习的技能是？
SQL和PLSQL语言、Oracle 数据存储结构、Oracle 实例组建、Oracle 备份与回复、性能优化诊断和调整等。
那ACCP课程体系又是如何的呢？我们来看一下ACCP课程的体系表，这张表在每本书，每个课程开始时，教员都会为我们分析本门课程在整个ACCP体系中的地位！

可以看到，ACCP课程分为三个学期，从上到下分为为：第一学年两个学期(S1,S2)，第二学年一个学期(Y2)，内容主要分为SQL Server数据库技术，Java方向以及 .Net C#方向。当然，福州三拓已经不教授的《职业导向训练》就不说了。

很明显，这两门课程在内容上是完全不一样的，那么，以WDP课程来替换原先的ACCP课程学员为什么会同意呢？


  1. 这是Oracle课程首次在海西推广WDP课程，落户福建，与福建信息产业厅以及福州三拓合作。这是政府支持！

  2. 首先，在新的课程开始之前，我们这些原先即将学习Y2课程接受了一场福州三拓安排的课程说明会，课程中主要说明了Oracle课程的优势性以及SQL Server作为商业软件的封闭，性能及功能的低下，还有就是教员方面隐晦的告诉我说，ACCP认证在外面不被承认。另外，学校称Oracle考虑到我们原先的课程中有JAVA和C#内容，Oracle特别为我们增加了这两门课程。但这个说法很明显不靠谱，因为福州三拓的新学员，也就是说本来应该上S1课程的学员的课程也被改成”WDP”课程了。当然，这些新生绝大部分是不知道情况的。他们可能不懂什么是SQL Server，什么是Oracle，他们只要知道北大青鸟，然后又被告诉有一个比北大青鸟更好的厂商就可以了。这是课程质量保证

  3. 原先我们这些准备上Y2课程的学员都已经提早交了一万两千两百六十六块钱（容许我记错一两百块钱），福州三拓告诉我们学费是一样的。这是学费保证！


我们看到，有了政府，课程质量，学费三个方面的保证后，很多学员没有查询更多的相关资料，误以为在课程质量方面是有保障的，答应使用WDP课程内容。但是，经过几位只相信自己看到、查到的学友查询，福建信息产业厅称并没有和福州三拓合作推广WDP课程这一事宜，另外，WDP课程的市场价格在八千元左右。而福州三拓的WDP课程是一万两千多，另外竟然可以选择的考OCP认证，要知道，OCP认证可是WDP的最终目标。那如果加上福州三拓给我们的OCP认证费用3000元。那就是一万五千多。为何都是WDP课程，费用为何差了一倍之多。

我总结出了福州三拓在这次WDP课程更换中的各种疑点：

  1. 福建三拓称与福建信息产业厅以及Oracle合作，但信息产业厅得到的结果是：不存在这个WDP推广合作关系。另外，我与Oracle WDP课程的中国区联系人（https://workforce.oracle.com/pls/wdp_ekit/rd_psp_contact1）：Jerry.yu@oracle.com +86-21-23023237 联系后，得到了一个非正面的回答：Ivy，Pls help check more details.

  2. 从我们目前我们正在学习的ASP.NET课程的PPT来看，依然是北大青鸟的ACCP课程内容。

  3. 学员被要求使用WDP课程后，福州三拓与学员签定了一份合同，当合同内容对Java与c#内容一字不提。

  4. 福州三拓称Oracle首次在福州招生，但在http://www.sato.com.cn/_center/370.htm看到，福州三拓在2003年已经加入ORACLE 公司战略合作联盟 WDP( Oracle Workforce Development Program)
