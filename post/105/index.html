一起来加速IE6的灭亡

2009年09月26日

Windows XP的成功与长寿，成功造就了IE6的霸主地位，即使IE7,IE8的推出，依然无法撼动IE6的地位。

你会发现 IE 各版本的占有率和 Windows 的占有率成正比,可以说，Vista的失败令IE7无法威胁IE6在全球范围内的领先占有率地位。新一代的操作系统Windows 7将在10月22日正式发售，Windows 7系统集成了IE8，Win7在正式发售前已经获得了非常空前的好评。分析称IDC称，到2013年，在微软销往企业用户的系统中，Windows 7将高达95%。但请注意，这只是销往企业用户的系统占有率，所以，XP仍将占有相当大的市场份额。

靠新系统来更改浏览器市场的占有率是不现实滴，人人都应该出一份力，加速IE6的灭亡。

对于主题开发者来说,可以在自己的作品中加入提示窗口或提示安装Google Chrome Frame插件.利人利己啊.
