我的第二个Wordpress Theme：iBluety

2011年06月22日

这是我制作的第二个 Wordpress 主题，2010.07月左右开始制作，一直准备发布，但这一拖就是近一年的时间。光阴似箭催人老啊。

相比制作第一个主题 Simger 时，对 html/css 的掌控进步很多，当然主题各个方面都完善很多。

博客已经荒废多时，上一次认真地写一篇Post已经不记得是什么时候了，近期又考虑着将越来越臃肿的 Wordpress 换掉。要学的东西太多，没有太多时间整理这个主题，仓促打包发布，所以你可能需要自己动手修改一番。

###支持的功能：

  • 支持后台自定义菜单功能，不支持下拉菜单
  • 自带分页导航
  • 自带存档页( archives.php )和链接页( links.php )
  • 替换搜索功能为Google自定义搜索，（请创建你自己的 Google 自定义搜索，然后替换 cse.php 第77行的那串数字，接着在后台创建一个页面，使用CSE这个模版，自定义地址为search，如果自定义地址不是这样的格式，请修改header.php文件里<form method="get" id="searchform" action="search">的action的值。）
  • 实现头像缓存功能。（请在wp-content下面建立了avatar目录）

###需要提醒：

  • 大概、应该、也许、可能是做到了兼容各版本浏览器。有什么bug欢迎反馈。
  • 侧栏有几个社交图标，像微博呀，Flickr呀，我注释掉了，如果你需要，打开 sidebar.php，去掉12-17行的注释，更换成自己的地址，图标 PNG源图 在主题的 png 文件夹里
  • style.css 被我压缩过，格式良好，注释齐全的请看 style_all.css
    下载地址：Google Code

有相关的使用问题，修改问题，可以到 WOPUS问答 提问，我会定期去看看滴~~
