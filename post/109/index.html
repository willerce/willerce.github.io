《使用Java语言理解程序逻辑》课程结束

2009年11月05日

11月3号，我在北大青鸟学习的ACCP5.0软件开发初级程序员系列课程中的《使用Java语言理解程序逻辑》课程结束了。

这本书涉及的内容比较简单,主要是一些基础知识,比如变量,判断,循环,数组等.提到了面向对象的编程思想,但并没有深入.

一周多月的学习,我也开始对北大青鸟的授课有了比较全面的认知了.总的来说,北大青鸟的课程还是比较不错的,不管从授课方式,还是书本内容上,都是比较适合零基础的学员的.但可能因为职业教育的时间等问题上考虑,它的内容点并不是那么全面,只是单纯为企业输送代码工罢了.我的同学,很大一部分可能到现在为止还没明白什么是JDK.

我所在的校区,一个班级一般是25名左右,我感觉人数还是有些多,很多玩意儿,全班只是一部分人懂了,但课程时间已经完了,教员也就只是把代码拷一份就OK了.所以,在北大青鸟的学习过程,自学能力的提升还是相当重要的.预习工作一定要做好!

最后贴一段代码(骑士飞行棋中的一小段)

        
//传参数，判断每格的样式
public String getGraph(int index,int i,int player1,int player2) {
    String graph = null;
    if(player1 == i && player2 == i) {
        graph = "@@";
    }else if( player1 == i) {
        graph = "A";
    }else if (player2 == i ) {
        graph = "B";
    }else {
        switch(index) {
            case 0:
                graph = "∷";
                break;
            case 1:
                graph = "★";
                break;
            case 2:
                graph = "◎";
                break;
            case 3:
                graph = "■";
                break;
            case 4:
                graph = "〓";
                break;
        }
    }
    return graph;
}
